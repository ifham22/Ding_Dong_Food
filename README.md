## Description

Ding_Dong_Food is a comprehensive food delivery and restaurant management system developed as a lab project for SWE 4301. It allows users to browse restaurants, view menus, place orders, and manage payments. Restaurants can register, manage their menus, and track orders. The system includes both a backend server written in Java and a web-based frontend.

## Features

- **User Management:** Register and manage user accounts.
- **Restaurant Registration:** Restaurants can register and provide details.
- **Menu Management:** Add, update, and manage menu items with quantity tracking.
- **Order Placement:** Users can search for restaurants, view menus, and place orders.
- **Payment Processing:** Integrated payment processing for orders.
- **Search Functionality:** Search for restaurants based on criteria.
- **Scheduling:** Manage order scheduling.
- **Web Interface:** User-friendly web interface for browsing and ordering.
- **Data Persistence:** Uses CSV files for storing data (users, restaurants, menus, orders).

## Technologies Used

- **Backend:** Java (with classes for server, menu, orders, etc.)
- **Frontend:** HTML, CSS, JavaScript
- **Database:** CSV files for simplicity
- **Server:** Java-based server

## Installation and Setup

1. Ensure you have Java installed on your system.
2. Clone the repository: `git clone https://github.com/ifham22/Ding_Dong_Food.git`
3. Navigate to the project directory.
4. Run the `.\run.bat` file to start the server.
5. Open `html/index.html` in a web browser to access the frontend.

## Usage

- Start the server using `.\run.bat`.
- Access the web interface to register users, browse restaurants, place orders, etc.
- Restaurants can manage their data through the backend.
